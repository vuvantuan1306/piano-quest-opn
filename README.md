# Piano Quest

Piano Quest is a simple Web3-inspired piano game built for OPN Builders Season 1.

Users can click piano keys, play notes, earn points, and experience the first demo version of a rhythm-based quest game on OPN Chain.

## Features

- Interactive piano keys
- Score system
- Last played note display
- Clean Web3-style interface
- Built with React and Vite

## Tech Stack

- React
- Vite
- JavaScript
- CSS
